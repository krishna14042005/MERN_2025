what is strictmode?

StrictMode is a React component used to identify potential issues and bugs in React applications. It provides warnings and alerts that Helps you prevent some side effects by identifying potential risks. As the documentation says, strict mode is development oriented so you don't need to worry about it impacting on your production build.
